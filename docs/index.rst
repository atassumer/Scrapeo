Scrapeo  Alpha 0.1 documentation
==========================

Auteur : Giloux

Install on Ubuntu
-----------------
::

   sudo apt-get install python-setuptools
   sudo easy_install django
   sudo easy_install gevent

Init database
-------------
::

   python manage.py syncdb

Launch developement server
--------------------------
::

   python manage.py runserver
